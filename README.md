# LOST.DIR-Recover

small program for recover file without exntention that located at LOST.DIR *(android sucks)*

i just made it for my old memory


## usage
- go [releases](https://github.com/Mystika/LOST.DIR-Recover/releases) and download latest version.
- run program in shell(command prompt) like this `LOSTDIRRecover.exe {LOST.DIR PATH} {NEW FILE DIRECTORY PATH}`
  (if you don't get this shit, just see example)
- done
